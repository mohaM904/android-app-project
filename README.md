SafeWalk is a mobile application designed to improve pedestrian safety by detecting unsafe smartphone usage while walking. The application utilizes built-in smartphone sensors, including the accelerometer and device orientation (and optionally the camera), to monitor user movement and phone usage behavior in real time.
When the system detects that the user is walking while actively engaging with their phone (e.g., looking at the screen), it triggers an immediate alert through vibration, sound, or on-screen notification. This helps reduce the risk of accidents caused by distracted walking.
The application operates entirely offline, ensuring accessibility and low resource consumption. Its lightweight design makes it suitable for everyday use, particularly among students and urban pedestrians.
